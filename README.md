#  Football Match Prediction & Value Betting Meta-Model

Ένα σύστημα πρόβλεψης ποδοσφαιρικών αγώνων για τα 5 κορυφαία ευρωπαϊκά πρωταθλήματα (Αγγλία, Ισπανία, Ιταλία, Γερμανία, Γαλλία). 

Το σύστημα συνδυάζει παραδοσιακά στατιστικά μοντέλα (Elo Ratings, Poisson Distribution) με σύγχρονους αλγορίθμους Μηχανικής Μάθησης (XGBoost) για να εκτιμήσει τις πραγματικές πιθανότητες των αγώνων και να τις συγκρίνει με τις αποδόσεις των στοιχηματικών εταιρειών.

---

## Δυνατότητες Συστήματος

* **Αυτόματη Λήψη Δεδομένων:** Αντλεί και συγχρονίζει αυτόματα ιστορικά δεδομένα αγώνων (2012-σήμερα) και αποδόσεις bookmakers.
* **Στατιστικός Πυρήνας (Statistical Core):** * Υπολογισμός δυναμικότητας ομάδων μέσω **Elo Ratings** (με προσαρμογές για πλεονέκτημα έδρας και διαφορά τερμάτων).
  * Εκτίμηση Expected Goals (xG) μέσω κατανομής **Poisson** με ενσωματωμένη διόρθωση **Dixon-Coles** για μεγαλύτερη ακρίβεια σε αποτελέσματα χαμηλού σκορ.
* **Meta-Model Μηχανικής Μάθησης:** Χρήση του αλγορίθμου **XGBoost** για τον εντοπισμό μη-γραμμικών μοτίβων και την ελαχιστοποίηση του σφάλματος πρόβλεψης.
* **Βαθμονόμηση (Calibration):** Εφαρμογή **Temperature Scaling** για τη διασφάλιση ότι οι προβλεπόμενες πιθανότητες ανταποκρίνονται στα πραγματικά ποσοστά επαλήθευσης.
* **Προσομοίωση Στοιχήματος (Backtesting):** Αναλυτική αναφορά κερδοφορίας (ROI, Net Profit, Hit Rate) ανά κατηγορία στοιχήματος (1, X, 2).

##  Αρχιτεκτονική (Δομή Αρχείων)
main.py: Ο κεντρικός ενορχηστρωτής. Ενώνει το Poisson/Elo με την αγορά και εκπαιδεύει το XGBoost.

setup.bat: Αυτοματοποιημένο script εγκατάστασης.

requirements.txt: Λίστα με τις εξαρτήσεις της Python (pandas, xgboost, scikit-learn κ.λπ.).

src/

update_data.py: Διαχειρίζεται τα web requests για τη λήψη των CSV.

data_processing.py: Καθαρισμός δεδομένων και αφαίρεση γκανιότας (overround) από τις αποδόσεις.

elo.py: Ο αλγόριθμος βαθμολόγησης Elo.

poisson_model.py: Ο υπολογισμός Expected Goals και scorelines (Dixon-Coles).

calibration.py: Η λογική του Temperature Scaling.

metrics.py: Συναρτήσεις αξιολόγησης του μοντέλου.
